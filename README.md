## üöÄ Getting Started

### For Developers

1. **Clone our repositories**
   ```bash
   git clone https://github.com/auroraio/aurora-core.git
   git clone https://github.com/auroraio/aurora-ai.git
   git clone https://github.com/auroraio/aurora-web.git
   ```

2. **Set up development environment**
   ```bash
   # Install dependencies
   npm install
   # or
   pip install -r requirements.txt
   ```

3. **Run locally**
   ```bash
   npm run dev
   # or
   python app.py
   ```
